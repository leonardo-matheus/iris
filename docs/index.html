"Todo dia era a mesma rotina..."

Como desenvolvedor senior, eu chegava ao trabalho e precisava executar uma sequencia de comandos no terminal para iniciar meu ambiente de desenvolvimento: 

# Terminal 1: Gateway
cd C:\projetos\gateway
npm install
npm run dev

# Terminal 2: Dashboard
cd C:\projetos\dashboard
yarn install
yarn dev

# Terminal 3: API
cd C:\projetos\api
docker-compose up

# Terminal 4: Microservicos
cd C:\projetos\services
cargo run

Eram 4 terminais, 12 comandos, e 5 minutos perdidos toda manha. Sem contar quando precisava reiniciar algo...

Foi entao que nasceu o Iris - um hub centralizado que executa todos esses comandos com um unico clique, mantem controle do que esta rodando, e me permite parar/reiniciar qualquer aplicacao instantaneamente.


Agora, em vez de 5 minutos, levo 5 segundos.
